# BenLang Dokumentation

Willkommen zur BenLang-Dokumentation! BenLang ist eine Programmiersprache, die speziell für Kinder entwickelt wurde, um spielerisch Programmieren zu lernen.

## Inhaltsverzeichnis

1. [Erste Schritte](01-erste-schritte) - Starte hier, um BenLang kennenzulernen
2. [Variablen](02-variablen) - Lerne, wie du dir Dinge merkst
3. [Zeichnen](03-zeichnen) - Male bunte Formen auf den Bildschirm
4. [Steuerung](04-steuerung) - Steuere dein Spiel mit Tastatur und Maus
5. [Schleifen](05-schleifen) - Wiederhole Dinge automatisch
6. [Bedingungen](06-bedingungen) - Lass dein Programm Entscheidungen treffen
7. [Figuren](07-figuren) - Erstelle bewegliche Spielfiguren
8. [Spiel erstellen](08-spiel-erstellen) - Baue dein eigenes Spiel
9. [Befehlsreferenz](09-befehlsreferenz) - Alle Befehle auf einen Blick

## Was ist BenLang?

Mit BenLang kannst du:
- Eigene Spiele erstellen
- Figuren auf dem Bildschirm bewegen
- Mit Tastatur und Maus steuern
- Punkte zählen und anzeigen

## Schnellstart

```bash
./benlang ./beispiele/snake
```

Viel Spass beim Programmieren!
